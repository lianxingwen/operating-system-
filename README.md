# 简单文件系统模拟器

这是一个用C语言实现的简单文件系统模拟器，支持基本的文件和目录操作。

## 功能特性

- 用户登录/登出系统
- 创建和删除文件
- 创建和切换目录
- 文件读写操作
- 目录浏览
- 权限控制

## 编译和运行

```bash
# 编译
make -f Makefile_complete

# 运行
./filesystem
```

## 使用说明

### 1. 登录系统
```
login <用户ID> <密码>
```

可用用户账户：
- 2116 (密码: dddd)
- 2117 (密码: bbbb) 
- 2118 (密码: abcd)
- 2119 (密码: cccc)
- 2220 (密码: eeee)

### 2. 基本命令

- `help` - 显示帮助信息
- `dir` - 显示当前目录内容
- `mkdir <目录名>` - 创建目录
- `cd <目录名>` - 切换目录
- `create <文件名>` - 创建文件
- `open <文件名> <模式>` - 打开文件 (1=读, 2=写)
- `write <文件描述符> <内容>` - 写入文件
- `read <文件描述符> <字节数>` - 读取文件
- `close <文件描述符>` - 关闭文件
- `delete <文件名>` - 删除文件
- `logout` - 登出当前用户
- `exit` - 退出系统

### 3. 使用示例

```bash
# 登录
login 2116 dddd

# 查看目录
dir

# 创建文件
create test.txt

# 打开文件进行写入
open test.txt 2

# 写入内容 (假设返回的文件描述符是0)
write 0 Hello,World!

# 关闭文件
close 0

# 打开文件进行读取
open test.txt 1

# 读取内容 (假设返回的文件描述符是0)
read 0 20

# 关闭文件
close 0

# 创建目录
mkdir mydir

# 切换目录
cd mydir

# 返回上级目录
cd ..

# 删除文件
delete test.txt

# 登出
logout

# 退出系统
exit
```

## 技术实现

- **文件系统结构**: 使用inode结构管理文件和目录
- **磁盘模拟**: 使用单个文件模拟磁盘存储
- **内存管理**: 实现inode缓存和文件描述符管理
- **权限控制**: 基于用户ID和组ID的权限检查
- **目录管理**: 支持层次化目录结构

## 文件结构

- `complete_filesystem.c` - 完整的文件系统实现
- `Makefile_complete` - 编译配置文件
- `README.md` - 说明文档

## 注意事项

1. 文件系统数据存储在当前目录的 `filesystem` 文件中
2. 首次运行会自动创建文件系统
3. 程序退出时会自动保存所有数据
4. 支持多用户并发访问（在单进程内模拟）

## 限制

- 文件大小限制：每个文件最多6个数据块
- 目录项限制：每个目录最多32个条目
- 用户数限制：最多10个并发用户
- 文件描述符限制：每个用户最多16个打开文件
